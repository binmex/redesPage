Instalación servidor HTTP

Instalacion de Apache

Para instalar el servidor HTTP, primero debes descargar el paquete apache2 con el comando apt-get install apache2

Después de haber instalado el paquete, utiliza el comando service apache2 status para verificar el estado actual del servicio de HTTP

Verificamos que este funcionando el servicio

Ahora que has confirmado que el servicio está en funcionamiento, puedes usar un navegador web para acceder a la carpeta de inicio del servidor Apache. Esta se encuentra en la dirección IP del servidor que has asignado, que en este caso es 192.168.1.1. También puedes utilizar el nombre de dominio de tu servidor para acceder a él, que en este caso es robinaguilar.co

Teniendo esto listo podemos pasar a configurar el servidor FTP.
