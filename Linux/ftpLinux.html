FTP - Linux

Instalación y configuración de FTP.

Instalamos FTP

Para instalar el servidor SMTP, primero debes descargar el paquete vsftpd con el comando apt-get install vsftpd.

configuramos el archivo /etc/vsftpd.conf

proceder a configurar el archivo vsftpd.conf. En este archivo, debes descomentar la línea que dice "write_enable=yes"

Después de haber revisado el estado del servicio, utiliza el siguiente comando para habilitar el tráfico entrante por el puerto 21 de FTP en caso de que el firewall lo esté bloqueando. Usamos el siguiente comando: sudo iptables -A INPUT -p tcp --dport 21 -j ACCEPT una vez finalizada esta parte podemos pasar con el servidor SMTP
