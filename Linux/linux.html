Debian 2012

Configuración de Debian

Configurar Hostname

Después de instalar Debian 12, inicia sesión con la cuenta root para evitar tener que conceder permisos de administración cada vez que necesites realizar una acción que requiera privilegios elevados. Una vez iniciada la sesión, realiza una comprobación de los nombres de host y los archivos hosts que se hayan configurado durante la instalación. Para verificar el nombre de host, puedes utilizar el siguiente comando en la terminal: 'hostname' Este comando te mostrará el nombre de host actualmente configurado en el sistema.

Verificar archivo de hostname

Verificamos el archivo de configuracion de hostname, para esto utilizamos el siguiente comando: "nano /etc/host", la configuracion debe ser como la siguiente imagen. cambianmos "robinaguilar.edu.co" por el dominio deseado

Configurar IP del servidor.

Luego, procede a configurar la dirección IP del servidor. Para esto, primero desactiva el servicio NetworkManager en caso de que esté activado, para evitar complicaciones en la asignación de direcciones IP estáticas.

Si esta activo el NetworkManager.

Si el servicio NetworkManager está activo, utiliza los siguientes comandos: 'systemctl stop NetworkManager' y 'systemctl disable NetworkManager'. Estos comandos detendrán y desactivarán el servicio NetworkManager, asegurando que no interfiera con la configuración de la dirección IP estática.

Consultamos los adaptadores de red

Una vez confirmado que NetworkManager está desactivado, puedes proceder a configurar la dirección IP del servidor. Para ello, primero utiliza el comando ls /sys/class/net para visualizar los nombres de los adaptadores de red disponibles en tu sistema. Es importante tener en cuenta que deberías tener al menos dos adaptadores de red en la máquina virtual: uno configurado en modo puente (bridged) para acceder a Internet y otro configurado en modo red interna (internal network) para servicios como DNS, DHCP, etc. en nuestro caso el adaptador enp0s8 sera el que usaremos para todo la configuracion de servicios.

Configuramos IP

Ya que conoces el nombre de tus adaptadores de red, primero crea una copia del archivo de configuración "interfaces" que se encuentra en /etc/network/. Luego, procede a configurar el adaptador con la dirección IP que desees. En este caso, la dirección IP es 192.168.1.1.

Edebemos usar los siguientes comandos, cp interfaces interfaces.original , despues nano interfaces

Reiniciamos sistema

Después de realizar la configuración, utiliza el comando "reboot" para reiniciar el servidor o simplemente usa el comando "systemctl restart networking". Luego, realiza pruebas con el comando "ip a" para verificar que la configuración se haya aplicado correctamente. Para reiniciar el sistema usamos el comando sudo reboot una vez se inicie el sistema entramos a consola y colocamos el siguiente comando ip a

Ya con esto se finaliza la instalación de Debian 12 en una máquina virtual de VirtualBox. Ahora se puede proceder a la configuración de los diferentes servidores y servicios necesarios para su funcionamiento en una red de dispositivos.
