Instalación y configuración de DHCP

Instalar Servidor DHCP

Para instalar el servidor DHCP, primero debes descargar el paquete isc-dhcp-server con el comando "apt-get install isc-dhcp-server".

copia de seguridad

Después de instalar el paquete, primero realiza una copia de seguridad de la configuración original del archivo isc-dhcp-server que se encuentra en el directorio /etc/default.

Modificar archivo isc-dhcp-server.

Ahora que has creado una copia del archivo de configuración original por si surge algún problema, puedes proceder a configurar el archivo isc-dhcp-server. En este archivo, especificarás las interfaces IPv4, en donde debes poner el nombre de la interfaz donde se configuró la IP (en este caso 192.168.1.1).

Configurar rangos del servidor

Después de hacer la configuración de isc-dhcp-server, primero realiza una copia de seguridad de la configuración original del archivo dhcpd.conf que se encuentra en el directorio /etc/dhcp.
