Servidor DNS

Instalación y configuración de servidor DNS.

Instalar bind9

Para instalar el servidor DNS, primero debes descargar los paquetes bind9 y dnsutils utilizando el comando sudo apt-get install bind9".

Después de instalar los paquetes, primero realiza una copia de seguridad de la configuración original del archivo named.conf.local que se encuentra en el directorio /etc/bind.

Configuracion de zonas.

Ahora que has creado una copia del archivo de configuración original por si surge algún problema, puedes proceder a configurar el archivo named.conf.local. En este archivo, especificarás la zona de búsqueda directa y la zona de búsqueda inversa, lo cual ayudará a resolver de forma adecuada el nombre de dominio que has elegido.
