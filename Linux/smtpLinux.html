SMTP - Linux

Instalación y configuracion de servidor SMTP

Instalamos POSTFIX.

Para instalar el servidor SMTP, primero debes descargar el paquete postfix con el comando "apt-get install postfix". sudo apt install postfix Cuando estés instalando Postfix, aparecerá un asistente de instalación en el que debes seleccionar la opción "Sitio web" y especificar el dominio del servidor, que en este caso es luis.com.co. Después de esto, la instalación del servidor continuará.

Realizar copia de seguridad

Después de instalar el paquete, primero realiza una copia de seguridad de la configuración original del archivo main.conf que se encuentra en el directorio /etc/postfix. cp /etc/postfix/main.cf /etc/postfix/main.cf.copy

Una vez que hayas creado una copia de seguridad del archivo de configuración original por si surge algún problema, puedes proceder a configurar el archivo `main.conf`. En este archivo, debes configurar los valores de `myhostname` para que coincida con el host de correos que creaste en la configuración del DNS, en este caso se llama correo.robinaguilar.co. También debes revisar la variable mydestination para asegurarte de que contenga los dominios correctos. Además, agrega la dirección IP del servidor en mynetworks y establece la variable home_mailbox = Maildir/.

Agregamos usuarios

Después de haber revisado que las configuraciones se hayan aplicado correctamente, utiliza el comando `adduser` para crear un nuevo usuario con el cual intercambiar correos.

Instalamos Dovecot

apt-get install dovecot-core dovecot-imapd

Después de instalar los paquetes, primero realiza una copia de seguridad de la configuración original de los archivos 10-ssl.conf, 10-auth.conf y 10-mail.conf que se encuentran en el directorio /etc/dovecot/conf.d. Luego procede a configurar los archivos según se muestra a continuación.

Thunderbird

Para el envío de correos podemos usar un cliente Telnet y probar el envío de correos por consola, aunque existe otra opción, la cual es usar las herramientas Thunderbird para realizar la interacción de email más amigable y funcional.

Configuramos Thunderbird

Luego de haber terminado la configuración de Dovecot, reinicia el servicio con el comando service dovecot reload. Después de reiniciar el servicio, instala Thunderbird e inicia sesión con las cuentas que creaste para enviar correos de prueba.
