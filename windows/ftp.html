FTP

Instalación y configuración de FTP.

Instalamos FTP

Vamos al asistente de roles y características, buscamos la sección de Servidos (IIS), desplegamos el selector y seleccionamos el apartado "Servidor FTP".

Verificar el proceso de instalación.

Le damos siguiente - siguiente hasta llegar al apartado de instalación, verificamos los servicios que se van a instalar y le damos clic en instalar.

Validar que esté funcionando el HHTTP

Al instalar el servidor ISS, por defecto se nos instalará lo necesario para el servicio de HTTP. Necesitaremos este apartado para poder terminar el FTP. Para lograr esto debemos acceder a un navegador y escribir la dirección IP del servidor, en nuestro caso 192.168.1.1.

Agregamos un sitio FTP.

Para agregar un sitio FTP, nos vamos en al apartado de herramientas. Buscamos la sección de administrador de herramientas de Internet Information Services (IIS). Una vez allí, desplegamos el servidor, la carpeta de sitios y damos clic derecho sobre la carpeta y damos clic en agregar sitio FTP.

Configuramos el servicio de FTP.

En este apartado debemos indicar qué seguridad deseamos, si anónima (no pide credenciales) o básica (pide usuario y contraseña), además le asignamos permisos, en este caso leer y escribir.

Validamos su funcionamiento.

Finalmente, nos dirigimos a un navegador, y debemos escribir ftp://(la dirección del servidor), en nuestro caso FTP://192.168.1.1. De esta manera nos mostrará la página FTP, que acabamos de configurar.
