Active Directory y DNS

Instalación y configuración de Active Directory y DNS.

Configurar IP estática

Para establecer una dirección estática, tenemos que ir a propiedades de Ethernet, buscamos nuestro adaptador, damos clic derecho en las propiedades y buscamos protocolo IPv 4.

Verificar el proceso de instalación.

Debemos indicarle una dirección estática y asignarle una máscara de subred, en este caso nuestra dirección IP es. 192.168.1.1 y la máscara 255.255.255.0. En el servidor DNS dejamos la misma dirección IP.

Instalamos el servicio de Active Directory

Para esto vamos a agregar roles y características. En roles del servidor buscamos servicios de dominio Active Directory. Lo seleccionamos y le damos siguiente, siguiente e instalar.

Configuración de Active Directory

En el asistente para la configuración de servicios de dominio de Active directory, damos clic en agregar un nuevo bosque e indicamos nuestro dominio. En nuestro caso robinaguilar.co y damos lo siguiente.

Debemos escribir una contraseña de restauración.

Debemos asegurarnos de que sea una contraseña segura y que cumpla con las políticas de Microsoft. Debemos asegurarnos que esté marcada la casilla de servidor del Sistema de nombre de dominio (DNS).

Verificamos el nombre de dominio NetBIOS.

Nos pedirá que verifiquemos el nombre del dominio. Si todo está bien, le damos siguiente, siguiente.

Cerrar sesión.

Una vez terminamos de configurar el Active Directory, nos pedirá que cerremos sesión y con esto podremos agregar nuevos usuarios de Active Directory.

Buscamos Usuarios de Active Directory

Una vez volvamos a iniciar sesión, buscamos usuarios y equipos de Active Directory, en el menú de Windows Server.

Creamos usuarios.

¡En este apartado podemos crear unidades organizativas y usuarios, para esto damos clic derecho en alguna de las unidades organizativas existentes o creamos una nueva! ¡Una vez allí, nos pedirá que llenemos los datos para crear un nuevo usuario y listo! Tendríamos nuestro dominio y Active Directory configurados.
